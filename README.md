## 특징

- 본글에서 falsy값이 렌더링 되던 이슈 수정
- useEffect 기능 구현

## 로컬에서 실행하기

Clone the project

```bash
  git clone https://github.com/clianor/mini-react
```

Go to the project directory

```bash
  cd mini-react
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run start
```
